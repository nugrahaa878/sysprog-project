## Petunjuk
1. Simpan file ```install.sh``` dan ```control_audio.sh``` pada ~
2. Compile dengan menjalankan command ```chmod +x install.sh```
3. Tambahkan install.sh pada ~/bashrc dan jalankan  ```source ~/.bashrc ```
4. Jalankan ulang VirtualBox
